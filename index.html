Assignment1

Project Description

Assignment 1 is an overview of the basics for HTML, CSS, and how to structure a website for easy navigation and management.

  • Create a main page that acts as an ePortfolio with descriptions of our assignments.
  • Create a link to the first assignment.
  • Create a style sheet to alter the appearance of the ePortfolio page as well as the content in assignment 1.
  • Construct an About Me page that displays your full name, email address, and relevant information about your past, present, and future.
  • The About Me page also includes an image of yourself, list of favorite websites, and academic/professional achievements and goals.

Assignment2

Project Description

Assignment 2 is an introduction to form elements and basic functionality using JavaScript.

  • Create a form that a user can input various types of data into such as text, drop down selections, radio buttons, and check boxes.
  • Utilize the existing style sheet created in the previous assignment and update it as needed to style the form and all of the elements.
  • Create JavaScript functions that will allow the user to click a submit button to validate and process the data as well as a button to reset the form.
  • Once the form data is submitted provide feedback to the user to notify them that the form was submitted or to alert them if any of the data is invalid.
  • Create a link to the Flowchart that provides a graphical representation of the logic that is used to validate and process the form.

Assignment3

Project Description

Assignment 3 tests your ability to perform mathematical calculations with arrays in JavaScript

  • Create a form that a user can use to input a set of data delimited by commas.
  • Utilize the existing style sheet created in the previous assignment and update the styling with external frameworks such as bootstrap.
  • Create JavaScript functions that will utilize the data set entered by the user and perform various mathematical calculations and present the results to the user.
  • Provide the user with the ability to view three sets of test data so they can compare these results with the ones produced by the functions created in JavaScript.

Assignment4

Project Description

Assignment 4 introduces us to the HTML5 canvas

  • Create an HTML5 canvas and add objects to it.
  • Use the objects added to the canvas to show a visual representation of a concept.
  • Use JavaScript to position the objects and animate them on the canvas.
